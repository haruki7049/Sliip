# Examples

The `examples/` directory contains various sample programs demonstrating different features of Sliip:

- **hello-sliip.sliip** - Basic hello world program
- **arithmetic.sliip** - Working with numbers and numeric types
- **variables.sliip** - Variable definitions and references
- **functions.sliip** - Lambda functions and function composition
- **higher-order.sliip** - Higher-order functions (functions returning functions)
- **chaining.sliip** - Function chaining and composition
- **recursive-refs.sliip** - Recursive variable references
- **multi-param.sliip** - Functions with multiple parameters
- **closure.sliip** - Closures and captured environments
- **nested-calls.sliip** - Deeply nested function calls
- **type-annotations.sliip** - Type system and explicit type annotations

Run any example with:

```bash
stack run -- examples/<filename>.sliip
```
